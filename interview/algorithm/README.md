#### 贪心算法
贪心算法遵循一种近似解决问题的技术，期盼通过每个阶段的局部最优选择（当前最好的解），
从而达到全局的最优（全局最优解）。
贪心得到结果是一个可以接受的解，不一定总是得到最优的解
